This tiny open-source Chrome extension changes the appearance of standard boards on Chess.com website. Please note that this extension is just a UI transformer, so it does NOT violate any fair-play policy.

Features:
Replace the standard board with Matrix-style digital rain.
Change piece icons to letters.
Display move notation on the board.
